# 健身操运动陪伴系统

这是一个面向 GitHub 协同创作的项目起始仓库，用于沉淀“AI 健身操运动陪伴系统”的完整方案、需求与设计内容。

## 项目定位

本项目聚焦居家轻量化健身场景，融合智能训练、AI 陪伴、社交打卡、语音交互与多设备联动，打造更有趣、更智能的健身操陪伴体验。
