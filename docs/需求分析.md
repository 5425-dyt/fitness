# 需求分析

## 1. 用户痛点

- 居家健身缺乏持续动力；
- 用户在动作标准性和训练节奏上容易偏离；
- 缺少实时陪伴和轻量社交互动；
- 传统方案缺乏对身体状态的动态适配。

## 2. 核心需求

### 2.1 训练需求

- 支持个性化健身操计划；
- 根据疲劳度、心率、训练目标动态调整强度；
- 提供动作纠错和训练反馈。

### 2.2 陪伴需求

- 提供虚拟 AI 形象实时陪伴；
- 支持语音控制和轻量对话；
- 针对不同训练阶段提供鼓励与提醒。

### 2.3 社交需求

- 支持好友组队、打卡、PK；
- 提供多人联机共跳场景；
- 通过积分和等级形成正向激励。

## 3. MVP 范围

- 用户注册与个人画像；
- 基础训练计划生成；
- AI 陪伴展示与语音交互；
- 简化版社交打卡与积分体系。
